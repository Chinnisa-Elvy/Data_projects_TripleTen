# Superstore Profitability & Returns Analysis

## Overview
This project analyzes retail sales, profitability, and return behavior to identify profit drivers and operational inefficiencies.

## Business Objective
Support pricing, promotion, and inventory decisions by understanding profit patterns and product return drivers.

## Tools Used
- Tableau

## Key Analyses
- Analyzed profit and loss by product, region, and shipping mode
- Evaluated seasonal and geographic trends to assess advertising effectiveness
- Calculated return rates across products, customers, and locations

## Key Insights
- Identified high-margin and loss-making product categories
- Revealed customer segments and regions with elevated return rates
- Highlighted seasonal trends impacting profitability

## Deliverables
- Interactive Tableau dashboards
- Visual storytelling to support stakeholder decision-making
