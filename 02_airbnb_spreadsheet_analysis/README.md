# Airbnb Market & Revenue Analysis

## Overview
This project analyzes Airbnb listing and calendar data to understand pricing, demand, and revenue potential across different neighborhoods.

## Business Objective
Identify high-performing locations and property types to support short-term rental investment decisions.

## Tools Used
- Google Sheets
- Excel

## Key Analyses
- Cleaned and validated listing and calendar datasets
- Analyzed booking activity by neighborhood and property size
- Estimated monthly and annual revenue potential using pricing and availability data
- Conducted funnel, cohort, and retention-style analysis to assess customer behavior

## Key Insights
- Identified neighborhoods with the highest revenue potential
- Highlighted optimal property sizes associated with higher occupancy
- Revealed pricing patterns linked to sustained booking performance
