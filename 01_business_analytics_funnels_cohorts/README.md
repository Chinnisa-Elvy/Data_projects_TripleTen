## Overview
This project analyzes raw e-commerce event logs to understand how users move through the website and how well the platform retains customers over time.

## Business Objective
Identify conversion drop-off points and measure customer retention to support improvements in user experience and revenue performance.

## Tools Used
- Excel
- Google Sheets

## Key Analyses
- Built a three-stage conversion funnel using unique user counts
- Calculated total and step-by-step conversion rates
- Created acquisition cohorts based on first purchase month
- Calculated monthly retention rates using cohort analysis

## Key Insights
- Identified major user drop-off between product views and purchases
- Revealed declining retention after the first purchase month
- Provided clear metrics to support funnel optimization decisions
