# Shopify App Analysis

## Overview
This project analyzes Shopify App Store data to identify factors that influence app success, user engagement, and developer responsiveness.

## Business Objective
Evaluate app performance and review behavior to understand what drives higher ratings and engagement.

## Tools Used
- Power BI
- DAX

## Dataset Overview
The analysis used app metadata, categories, and user reviews scraped from the Shopify App Store.

## Key Analyses
- Measured app marketplace size and review trends over time
- Created weighted review metrics using DAX
- Compared ratings for apps with and without developer responses
- Evaluated developer responsiveness for high-review apps

## Key Insights
- Apps with active developer responses showed higher average ratings
- Weighted review metrics provided better quality signals than raw ratings
- Certain developers consistently demonstrated stronger engagement

## Deliverables
- Power BI report with KPIs and interactive visuals
