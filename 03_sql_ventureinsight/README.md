## Overview
This project analyzes a multi-table venture capital database to understand startup performance, funding trends, acquisition behavior, and investor strategies.

## Business Objective
Support investment decision-making by answering business questions related to startup success, risk, and fund activity.

## Tools Used
- SQL (PostgreSQL)

## Dataset Overview
The analysis was conducted using relational tables including companies, funds, funding rounds, investments, acquisitions, people, and education records.

## Key Analyses
- Measured startup failure rates to establish baseline risk
- Analyzed funding raised by U.S.-based news and media startups
- Evaluated cash-based acquisition trends during post-recession years
- Identified funding round volatility using aggregation techniques
- Classified venture funds into activity tiers using conditional logic

## Key Insights
- Revealed patterns in startup failure and funding concentration
- Highlighted differences in investment strategies by fund activity level
- Provided data-backed insights to support quarterly investment reporting
