# Gaming Console Sales Analysis

## Overview
This project analyzes historical sales data for discontinued gaming consoles to evaluate product performance and longevity.

## Business Objective
Support pricing and repair prioritization decisions using sales volume, lifespan, and revenue estimates.

## Tools Used
- Python

## Dataset Overview
The dataset includes console names, manufacturers, release years, discontinuation years, prices, and total units sold.

## Key Analyses
- Structured and displayed data using nested lists and loops
- Calculated total units sold across all consoles
- Created new metrics including product lifespan and estimated revenue
- Sorted and compared consoles based on performance metrics

## Key Insights
- Identified top-performing consoles by sales and revenue
- Highlighted differences between short-lived and long-lived products
- Demonstrated how custom metrics improve product comparisons
